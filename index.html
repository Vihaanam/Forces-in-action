Forces in action

About


Forces are pushes or pulls that are measured in newtons. Here are what forces can do:

  1. Make an object move. For example: we pedal and so the bicycle moves.


  2. Make a moving object stop. For example: we press the brakes to stop a moving car.


  3. Change the speed of a moving object. For example: we kick a ball and so it moves faster.


  4. Change the direction of a moving object. For example: in badminton, we change the direction in which the shuttle cock is coming.


  5. Change the shape of an object. For example: we mold clay into different shapes by applying force.
